hweden Tour Веб-приложение для бронирования туристических поездок и отелей.

Стек технологий: Frontend ReactReact, RouterHTML5 / CSS3Fetch, API. Backend Django, Django REST Framework, JWT (SimpleJWT), Database,SQLite. DevOps / Tools Git / GitHub Django Admin Panel

Основной функционал. Пользователи: Регистрация и авторизация JWT-аутентификация Личный кабинет пользователя Загрузка аватара Просмотр своих бронирований Поездки: Просмотр доступных поездок Поиск поездок по параметрам Бронирование поездок Отмена бронирования Акции (Offers): Просмотр акций Скидки на поездки Отображение старой и новой цены Бронирование поездок со скидкой Отели: Каталог отелей Детальный просмотр информации об отеле (всплывающее окно)

Административная часть. Административный функционал реализован через Django Admin Panel. Возможности администратора: Управление пользователями Управление поездками Управление акциями (Offers) Управление бронированиями ⚠️ Доступ в админку имеют только пользователи с ролью администратора. Кнопка «Админка» автоматически отображается в интерфейсе только у администратора. Доступ к админке http://127.0.0.1:8000/admin/ Логин и пароль тестового пользователя для каждой роли : admin@test.com / admin123, user@test.com / user123

Запуск проекта локально Backend

cd .\frontend\backend\

pip install -r requirements.txt

python manage.py migrate

python manage.py loaddata data.json

python manage.py runserver

Frontend

cd frontend

npm install

npm start
